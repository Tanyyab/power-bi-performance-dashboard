# Multi-Account Performance Dashboard — Power BI

## Overview
Centralised Power BI dashboard tracking conversion performance, 
account health, and pipeline metrics across 100+ concurrent 
global accounts across APAC, EMEA, and US markets.

## Business Problem
Senior stakeholders had no single view of account health, 
campaign performance, and pipeline data — reports were compiled 
manually, taking 5+ hours per week with no real-time visibility.

## What I Built
- Conversion tracking dashboard — form submissions, 
  button clicks, custom events
- Account health scorecard — KPIs per account, 
  flagging at-risk accounts
- Pipeline performance view — booking trends, 
  revenue forecasting, campaign ROI
- Customer behaviour trends — drop-off analysis, 
  engagement patterns

## Results
- Used by 20 internal stakeholders including senior management
- Saved 5 hours per week in manual report preparation
- Improved real-time visibility into 100+ account performance
- Reduced data discrepancies from ~20% to less than 5%

## Tools Used
Power BI | SQL | Looker Studio | Tableau | Excel | GA4

## Dashboard Sections
1. Executive Summary — top-line KPIs for leadership
2. Campaign Performance — conversion rates by channel
3. Account Health — per-account scorecard
4. Pipeline Tracker — booking and revenue trends
5. Customer Behaviour — funnel and drop-off analysis
